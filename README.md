## 🌟 Overview

Welcome to the source code of my personal portfolio website, hosted directly on GitHub Pages!

This portfolio has been carefully designed to highlight my professional journey, skills, and featured projects as a Mobile Developer. It focuses on providing a clean, distraction-free environment for HR recruiters and fellow developers.

🔗 **Live Website**: [seijind.github.io](https://seijind.github.io/)

## ✨ Key Features

- **Modern Aesthetic**: Deep "Sleek Dark" theme (`#0f172a` base) with indigo accents for a premium developer feel.
- **Glassmorphism UI**: Beautiful frosted-glass effects on cards and sections to create depth and focus.
- **Fully Responsive**: Crafted with modern CSS Grid and Flexbox to ensure a seamless experience on mobile, tablet, and desktop devices.
- **Performance Optimized**: Zero heavy CSS frameworks. Built entirely with Semantic HTML and pure vanilla CSS for lightning-fast load times.
- **Smooth Animations**: Integrated with AOS (Animate on Scroll) for subtle, non-intrusive entrance animations.

## 🛠️ Technologies Used

The site is deliberately kept lightweight and uses standard web technologies:

- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom design system utilizing CSS Variables (Custom Properties), Grid, and Flexbox.
- **JavaScript (Vanilla)**: For mobile navigation toggles and scroll effects.
- **AOS Library**: For the scroll animations.
- **FontAwesome**: For lightweight, scalable vector icons.
